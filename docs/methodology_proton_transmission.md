# Proton Transmission Methodology (Detailed)

## 1. 目的

本資料は `docs/report_proton_transmission.pdf` の内容を、実装観点で詳細化した技術メモです。  
対象コードは `src/proton_glass_transmission.py` です。

## 2. 問題設定

- 入力: 軌道環境の陽子微分フルエンス `g(E)`（4π空間）
- 遮蔽: ホウケイ酸ガラス厚 `t`（0.05, 0.10, 0.15, 0.20 mm）
- 出力: 透過後微分フルエンス `f(ε)`（2π空間）

太陽電池セル表面への入射を想定し、透過側半球でのスペクトルを求めます。

## 3. 数式の接続関係

### 3.1 飛程近似（式46）

`R(E) = A * E^a + B * E^b`

- `A = 202.0955`
- `a = 1.5279`
- `B = 14.7058`
- `b = 1.0166`

単位は `E: MeV`, `R: um` です。

### 3.2 透過条件（式50）

`R(ε) = R(E) - t/cosθ`

よって各 `(ε, θ)` ごとに必要入射エネルギー `E` は、

`R(E) = R(ε) + t/cosθ`

を満たす値として求めます。コードでは `brentq` による逆関数解法を使います。

### 3.3 ヤコビアン（式47, 48）

`f(ε) dε = g(E) dE`

`dE/dε = R'(ε) / R'(E)`

上式を使って、エネルギー変換時のスペクトル圧縮を補正します。

### 3.4 角度積分（式52）

`f(ε) = (1/2) * ∫ g[E(θ)] * R'(ε)/R'(E(θ)) * sinθ dθ`

- `sinθ dθ` は立体角重み
- `1/2` は 4π -> 2π 変換

数値実装では `θ in [0, π/2)` を離散化して和で近似します。

## 4. 実装アルゴリズム

1. `inputs/proton_environment_LEO600km.csv` から `E_env, F_env` を読込
2. `log10-log10` 線形補間で `g(E)` を構築（範囲外は0）
3. `inputs/range_borosilicate_glass.csv` から `E_ref, R_ref` を読込
4. `R_fit = R(E_ref)` と残差 `%` を計算し Fig.3 を出力
5. `ε` の対数グリッド（`1e-4`〜`1e3 MeV`）を生成
6. 各厚み `t` で `f(ε)` を角度積分し `outputs/transmitted_fluence.*` を出力

## 5. 主要パラメータ（デフォルト）

- `GLASS_THICKNESSES_MM = [0.05, 0.10, 0.15, 0.20]`
- `N_ENERGY_POINTS = 200`
- `N_THETA = 200`
- `EPS_MIN = 1e-4`
- `EPS_MAX = 1e3`

## 6. 出力の読み方

### 6.1 `fig3_range_fit.png`

- 上段: 参照飛程とフィット曲線がどれだけ一致するか
- 下段: 残差[%]でエネルギー帯ごとのズレを確認
- 用途: 飛程モデル妥当性の健全性チェック

### 6.2 `transmitted_fluence.png`

- 黒丸: 入力（4π）スペクトル
- 線群: 各厚みの透過後（2π）スペクトル
- 解釈: 厚み増加に伴い低エネルギー成分が削られ、積分フルエンスが減少

## 7. 既知の前提・制約

- 入射方向分布は等方（isotropic）仮定
- 入力スペクトルの補間範囲外は0扱い
- `θ = π/2` 近傍は数値安定性のため少し手前まで計算
- 多層遮蔽・異方入射・二次粒子生成は対象外
